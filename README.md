[pyql-weather]
=================

Yahoo Weather para python
------------------------------------------------------------

Instalación:
------------------------------------------------------------

1. Descarga el paquete y ejecutar:

```bash
python setup.py install
```

2. A disfrutar de las consultas del clima en Yahoo!!!!
